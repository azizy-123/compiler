# C/C++ Code Parser and Validator

A Python lexical analyzer and syntax parser for C/C++ code.

## Features
- Tokenizes C/C++ code into keywords, operators, identifiers, and literals
- Validates syntax (balanced delimiters, statement structure)
- Reports errors with line numbers

## Usage
Run the program, enter C/C++ code, type `END` to finish. Returns True/False for syntax validity.
